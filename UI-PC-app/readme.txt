This is apllication that runs on the PC and has next functionality:
 - get list of companies from the internet, (done)
 - send information about companies to the board with STM32 microcontroller, (done)
 - receive list of top/bottom compaties from the board with STM32 microcontroller. (done)
 
 Launch app from stock_exchange_helper.py
 Needed:
  - Python 3.10
  - Tkinter library
  - Installed fonts on your PC
  - BeautifulSoup4 library
  - requests library
  - lxml library
  - PySerial library
